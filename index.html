📖 Panduan Setup Backend (Google Apps Script)

Ikuti langkah ini sekali saja agar riwayat perhitungan tersimpan terpusat di Google Sheets (bisa diakses dari banyak perangkat):

  1. Buat Google Sheets baru, beri nama misalnya "Riwayat HD Anak".
  2. Buka menu Extensions → Apps Script.
  3. Hapus kode default, lalu salin-tempel seluruh isi file Code.gs yang disertakan (lihat file terpisah).
  4. Klik Deploy → New deployment → Web app.
  5. Pada "Execute as" pilih Me, pada "Who has access" pilih Anyone, lalu klik Deploy.
  6. Izinkan akses (Authorize access) saat diminta.
  7. Salin URL Web App yang muncul, tempel ke kolom "URL Web App" di atas, klik Simpan URL.

Sheet akan otomatis dibuatkan header kolom saat data pertama disimpan.
